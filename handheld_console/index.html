Guide

I nintendo-switch like made with HTML, CSS + JS

in brackets ("key") are buttons on a keyboard

Screen is filled with pixels that can be selected with Dpad (direction keys)

Power (1) - screen ON and OFF

Volume (2) - Hide/Show selected pixels

Left Trigger (Z) and Right Trigger (X) - Shift pre-set pattern left or right

Minus (Q) and Plus (E) - Change color of screen bg + selector border or screen foreground depending on A

Dpad (Arrow Keys) - Active pixel selector on the screen

Interaction Buttons

W (W) - Change selector Pattern (border radius)

D (D) - Load/unload pre-set patterns

S (S) - Adjust pulsing effect on selector(s)

A (A) - Toggle Color Switcher [-] [+] between Screen + Border and Pixel selector

Home (O) - Clear All Pixels

Rec (I) - Clears any saved pattern

Left (K) Analog Sticks - Save pattern.

Right (L) Analog Sticks - Load saved pattern.

Tips

The patterns can be stacked or combined to form more patterns
